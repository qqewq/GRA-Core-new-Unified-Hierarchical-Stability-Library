# GRA-Core-new: Unified Hierarchical Stability Library

Полная production-ready реализация методологии GRA (Gödel-Reductio ad Absurdum), вобравшая **103 исследовательских проекта** из экосистемы `qqewq`.

Библиотека даёт единую кодовую базу для построения самоисправляющихся иерархических систем с гарантией устойчивости.

**Ключевые модули:**
- Настоящие метрики противоречий («пена») для банков, дронов, LLM, биологии, субъективности
- Иерархическая структура с операторами подъёма и штрафами согласованности
- GRA-шаг через критику и пересмотр (не *0.9)
- Полный функционал J с градиентным спуском
- Проверка устойчивости (Φ=0, ΔΦ=0, Δ²Φ>0) и тест на возмущение
- Распределённый рой, военный GRA, рой здоровья
- Мультиверсное выравнивание и слияние
- Heisenberg-рассуждение
- Гибридный резонансный алгоритм
- Биологическое обнуление (старение, лекарства, клеточные двойники)
- Философские слои (субъективность, интересубъективность, гармонизация)
- Ранговая оптимизация (поиск оптимального N)
- Модель вечного сознания
- Кинематографическое обнуление сцен
- Мета-обнулёнка (трансфинитное ядро, квантовый наблюдатель)
- Прикладные эксперименты (Рамануджан, поэзия, эмоциональный ИИ)
- Цифровая валюта AALY, физический AI
- Систематизатор разрывов «теория ↔ код»
- Коннекторы к реальным системам

Все старые репозитории перечислены в `archive/README.md`.
